Нет DXF? Как конвертировать DWG ▾
LibreCAD — бесплатно, Windows / macOS / Linux
  1. Скачайте: librecad.org
  2. Откройте DWG: Файл → Открыть
  3. Экспортируйте: Файл → Экспорт → Экспорт как DXF
  4. Выберите формат DXF R12 или DXF 2000
ODA File Converter — бесплатно, пакетная конвертация
  1. Скачайте: opendesign.com
  2. Input folder: папка с DWG-файлами
  3. Output format: DXF, версия R2010
  4. Нажмите Start
AutoCAD / nanoCAD / BricsCAD: Файл → Сохранить как → тип файла DXF

Рекомендуется DXF R12 или R2010 для максимальной совместимости.
